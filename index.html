Καλώς ήρθες στο Coding Club!

Παρακάτω μπορείς να βρεις τις κατηγορίες δραστηριοτήτων μας. Διάλεξε αυτή που σε ενδιαφέρει:

Σχετικά με εμάς

Το Coding Club είναι μια κοινότητα φοιτητών του Τμήματος Πληροφορικής και Τηλεπικοινωνιών του Πανεπιστημίου Αθηνών που στοχεύει στη μάθηση, συνεργασία και ανάπτυξη των τεχνολογικών δεξιοτήτων. Αποτελούμαστε από φοιτητές που διψάνε για γνώση, μετατρέπουν τις ιδέες τους σε κώδικα, και πιστεύουν ότι τα καλύτερα projects γεννιούνται με καλή παρέα και ακόμα καλύτερο brainstorming!

Τι προσφέρουμε:

  • Εβδομαδιαίες συναντήσεις όπου το debugging γίνεται team sport.
  • Workshops που κάνουν το coding να μοιάζει με παιχνίδι.
  • Pair programming sessions γιατί δύο μυαλά σκέφτονται καλύτερα από ένα.
  • Networking με άλλους developers (και ανταλλαγή memes φυσικά!)
  • Project challenges που θα σε κάνουν να ξεχάσεις το sleep mode.
